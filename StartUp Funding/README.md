# üöÄ Startup Funding Dashboard

This Power BI dashboard provides an interactive visual overview of startup funding trends, investor activity, investment types, and city-wise funding distribution. It helps track how much funding has been raised, which cities attract the most investment, and which startups and industry verticals are performing the best.

---

## üîç Key Features

- **Dynamic filters** for:
  - Date Range
  - Investor Name
  - Investment Type (Series A, Series B, Seed Round, Angel, Debt Funding, etc.)
  - Startup Name
  - City Location
- **Total Funding KPI** showing total capital raised (`$126.44M`)
- **Funding Trend Line** showing month-wise funding activity
- **Investment Breakdown** using donut chart by investment type
- **Investment Type & Sub-Vertical Analysis** using treemap visualization
- **Interactive City Map** showing startup presence and funding distribution
- **Startup-wise Revenue Visuals** showing total revenue, investor name, and city location
- **Industry Vertical Analysis** showing funding allocated across business segments like:
  - Energy  
  - Food & Beverages  
  - Social Media  
  - Nanotechnology  
  - Compliance  
- Seamless navigation between overall summary and detailed breakdown pages

## ‚ñ∂Ô∏è How to Use This Dashboard

1. **Select the desired date range** using the date slider to filter funding events.
2. Use the **Investor Name**, **Investment Type**, **Startup Name**, and **City Location** filters to narrow down insights.
3. View the **Total Funding KPI** to quickly understand overall fundraising performance.
4. Analyze **funding trends over time** in the line chart labeled *Amount in USD by Date*.
5. Explore **funding distribution by investment type** through the donut chart.
6. Use the **treemap** to compare contributions of investment types and sub-verticals.
7. Examine the **map visualization** to see funding distribution across different locations.
8. Review **startup-level performance** with bar charts showing total revenue and investor contributions.
9. Switch between dashboard pages using navigation buttons for summary and detailed insights.
10. Hover on visuals to view **tooltips and detailed values** for deeper understanding.

These steps help you interact with the dashboard effectively and uncover meaningful insights from the data.
